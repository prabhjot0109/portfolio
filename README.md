## üì¶ Installation & Setup

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Local Development

1. **Clone the repository:**

   ```bash
   git clone https://github.com/prabhjot0109/prabhjot-portfolio.git
   cd prabhjot-portfolio
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the development server:**

   ```bash
   npm run dev
   ```

4. **Open your browser:**

   Navigate to [http://localhost:8080](http://localhost:8080)

### Build for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.

## üì± Usage

The portfolio includes the following sections:

- **Home**: Hero section with introduction and social links
- **About**: Personal background and professional summary
- **Skills**: Technical skills and competencies
- **Achievements**: Notable accomplishments and certifications
- **Projects**: Portfolio of completed projects with images and descriptions
- **Experience**: Professional work experience and roles
- **Contact**: Contact information and social media links

## üé® Customization

To adapt this portfolio for your own use:

1. **Personal Information**: Update details in component files (`Hero.tsx`, `About.tsx`, etc.)
2. **Projects**: Replace project images in `src/assets/` and update project data
3. **Styling**: Modify colors and themes in `tailwind.config.ts`
4. **Resume**: Update the PDF file in `public/resume.pdf`
5. **Social Links**: Change URLs in the Hero component
6. **Content**: Edit text content in respective components
