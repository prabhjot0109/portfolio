## 🛠️ The Tech Stack

- **Core**: [React 18](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Utilities**: [TanStack Query](https://tanstack.com/query) + [Zod](https://zod.dev/)
- **Infrastructure**: [Bun](https://bun.sh/) (Runtime) + [Vercel](https://vercel.com/) (Deployment)
